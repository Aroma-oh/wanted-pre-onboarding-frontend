## 커밋 메시지 컨벤션
* 예시: `feat: 로그인 기능 구현`

|커밋 유형|의미|
|:---:|:---:|
|init| 프로젝트 시작|
|feat| 기능 추가|
|style| 코드 포맷팅|
|refactor| 코드 리팩토링|
|chore| 패키지 매니저 및 그 외 기타 수정 ex) .gitignore|
|rename| 파일 또는 폴더 명을 수정하거나 옮기는 작업만인 경우 |
|remove|파일을 삭제하는 작업만 수행한 경우|
|setting|기본 세팅 변경의 경우|
